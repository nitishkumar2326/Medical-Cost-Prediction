Medical Cost Prediction using Machine Learning

Project Overview
This project focuses on predicting individual medical insurance charges based on demographic and lifestyle factors. The objective is to build accurate regression models that can estimate healthcare costs and analyze the impact of features such as age, BMI, smoking status, and region.

Models Implemented
Linear Regression was used as a baseline model to understand linear relationships between features and medical costs.
Random Forest Regressor was applied as an advanced ensemble technique to capture non-linear patterns and improve prediction accuracy.

Results
The Linear Regression model achieved an R² score of 0.78, indicating good performance.
The Random Forest model improved the performance significantly with an R² score of 0.85, demonstrating better generalization and accuracy.

Technologies Used
Python for implementation
Pandas and NumPy for data manipulation
Scikit-learn for machine learning models
Matplotlib for data visualization

Dataset
The dataset used is the Medical Cost Personal Dataset, which contains information such as age, gender, BMI, number of children, smoking status, region, and insurance charges.

Key Features of the Project
Data cleaning and preprocessing
Conversion of categorical variables into numerical format
Exploratory data analysis and visualization
Model training and evaluation
Comparison of multiple machine learning models

Outcome
The project successfully demonstrates how machine learning techniques can be used to predict medical costs. The use of Random Forest significantly improved prediction accuracy compared to Linear Regression, highlighting the importance of model selection in real-world scenarios.
